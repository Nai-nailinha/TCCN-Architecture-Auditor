# Modelo de Especialistas Internos do TCCN AI Studio

**Versão:** 0.1
**Status:** Oficial
**Categoria:** Operação
**Tipo de documento:** Modelo Operacional
**Fonte oficial:** Organização, responsabilidades e funcionamento dos especialistas internos do TCCN AI Studio.

---

# Objetivo

Este documento estabelece o modelo utilizado pelo TCCN AI Studio para organizar e utilizar especialistas internos durante a execução das demandas.

Sua finalidade é definir como as competências do estúdio são estruturadas, distribuídas e reutilizadas, preservando responsabilidade única, modularidade e colaboração entre especialistas.

Este documento define **como os especialistas funcionam**, não quais especialistas existem. A lista oficial de especialistas pertence ao **Catálogo de Especialistas Internos**.

---

# Finalidade dos especialistas

Os especialistas representam competências técnicas reutilizáveis.

Eles existem para:

* organizar responsabilidades;
* aumentar a qualidade das entregas;
* evitar sobreposição de funções;
* facilitar reutilização de conhecimento;
* permitir evolução independente das competências do estúdio.

Especialistas representam **capacidades**, e não personagens ou perfis humanos.

---

# Princípios de funcionamento

Os especialistas devem atuar conforme os seguintes princípios:

* responsabilidade única;
* colaboração entre competências;
* reutilização de conhecimento;
* independência entre especialistas;
* simplicidade operacional;
* adaptação ao contexto da demanda;
* decisões orientadas por evidências.

---

# Ativação dos especialistas

Os especialistas são ativados conforme a necessidade identificada durante a interpretação da demanda.

Nem toda solicitação exige a participação de todos os especialistas.

Sempre que possível, o estúdio deve utilizar apenas as competências realmente necessárias para atingir o objetivo da demanda.

---

# Responsabilidades

Cada especialista deve possuir responsabilidades claramente definidas.

Essas responsabilidades não devem duplicar funções atribuídas a outros especialistas.

Quando uma atividade envolver múltiplas competências, cada especialista contribui apenas dentro de seu domínio de responsabilidade.

---

# Colaboração

Especialistas podem colaborar entre si durante uma mesma demanda.

A colaboração deve ocorrer por composição de competências, preservando a autonomia de cada especialista.

Sempre que possível, informações reutilizáveis devem ser compartilhadas por meio das bibliotecas oficiais, e não por dependências diretas entre especialistas.

---

# Evolução

Novos especialistas podem ser incorporados ao estúdio quando:

* representarem uma competência claramente distinta;
* não duplicarem responsabilidades existentes;
* contribuírem para a evolução da arquitetura;
* respeitarem os princípios definidos pelo Manifesto.

Especialistas podem evoluir sem alterar a estrutura geral do modelo.

---

# Divergência entre especialistas

Durante uma mesma demanda, diferentes especialistas podem propor soluções distintas.

Quando houver divergência, a decisão deve considerar, nesta ordem:

1. o objetivo da demanda;
2. a autoridade técnica relacionada ao assunto;
3. os documentos oficiais da arquitetura;
4. a identidade do TCCN AI Studio;
5. os princípios de simplicidade, reutilização e coerência.

Quando a divergência envolver decisões estratégicas do estúdio, a definição final permanece sob responsabilidade da criadora do TCCN AI Studio.

---

# Integração com a arquitetura

O Modelo de Especialistas integra-se diretamente com:

* Modelo Operacional Enxuto de Demandas;
* Catálogo de Especialistas Internos;
* Fluxos de Produção;
* Bibliotecas Operacionais;
* Modelo de Controle de Qualidade (QA);
* Memória Permanente.

Os especialistas utilizam esses componentes como apoio, mas não os substituem.

---

# Observações

* Especialistas representam competências permanentes da arquitetura.
* O modelo define o funcionamento dos especialistas; o catálogo define quais especialistas existem.
* Especialistas podem ser adicionados ou evoluídos sem alterar a estrutura do modelo.
* A colaboração entre especialistas deve fortalecer a qualidade das entregas sem aumentar desnecessariamente a complexidade operacional.
